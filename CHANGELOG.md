# Change Log

All notable changes to the "relax-coding-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.1] - 2025-11-19

### Added
- **Relax Coding Light Theme:** A warm, paper-like light theme designed for long coding sessions.
- **Relax Coding Dark Theme:** A deep, cozy dark theme that minimizes eye strain.
- **High Contrast Variants:**
    - `Relax Coding Light High Contrast`: Sharper text and stronger borders on the warm light background.
    - `Relax Coding Dark High Contrast`: Bright text and grey borders on the warm dark background.
- **Multilingual Documentation:**
    - Added README translations for Japanese (`ja`), English (`en`), Korean (`ko`), Chinese (`zh`), Spanish (`es`), and Portuguese (`pt`).
- **MIT License:** Added license file.
